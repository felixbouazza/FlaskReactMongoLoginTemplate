# SeoIdeasApplication

An application who store all my seo script ideas with front-end dashboard

## Technologies

- __Back-end__ - FLASK
- __Front-end__ - REACT
- __Database__  - MONGODB

## Prerequisite

##### Python

```
python3 --version
// Python 3.8.5
```
##### Node
```
node --version
// v15.6.0
```
##### NPM
```
npm --version
// 7.4.0
```

##### NPX
```
npx --version
// 7.4.0
```

##### MongoDB
```
mongod --version
// db version v4.4.1
```

##### Postman

To test our API

##### Robo3t

Database Visualization

## Installation

##### Frontend

```
cd frontend/ && npm install 
```

##### Backend

```
cd backend/ 
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

##### Database

```
mkdir bdd

```

## Start

##### Frontend

```
cd frontend/
npm start
```

##### Backend

```
cd backend/ 
source venv/bin/activate
python app.py
```

##### Database

```
mongo --dbpath="bdd"

```
