# SeoIdeasApplication

An application who store all my seo script ideas with front-end dashboard

## Technologies

- __Back-end__ - FLASK
- __Front-end__ - REACT
- __Database__  - MONGODB

## Prerequisite

#### Python

```
python3 --version
// Python 3.8.5
```
#### Node
```
node --version
// v15.6.0
```
### NPM
```
npm --version
// 7.4.0
```

#### NPX
```
npx --version
// 7.4.0
```

#### MONGO
```
mongod --version
// db version v4.4.1
```

#### POSTMAN

To test our API

#### ROBO3T

Database Visualization

## Installation

#### Frontend

```
cd frontend/ && npm install 
```

#### Backend

```
cd backend/ 
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```
